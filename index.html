How to Use

For Desktop Host

  1. Click "Desktop Host" mode
  2. Copy the Device ID and Connection Code
  3. Click "Start Hosting" and grant screen permissions
  4. Share the codes with your mobile device

For Mobile Client

  1. Click "Mobile Client" mode
  2. Enter the host's Device ID and Connection Code
  3. Click "Connect to Host"
  4. Tap the screen to control the desktop
